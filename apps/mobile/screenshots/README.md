# Screenshots — kernschermen kandidaat-app

Vastgelegd via `apps/mobile/e2e/web-smoke.mjs` (Expo-web-render, iPhone-
viewport 390×844 @2x) tegen de **echte** lokale backend met geseede data.
Deze frames tonen de daadwerkelijke schermen; op een echt iOS-toestel rendert
de MatchShape-SVG ook de blob-vormen (op web-render is soms alleen het
scorecijfer zichtbaar).

| 04-onboarding-functie | Onboarding — functie & ervaring |
